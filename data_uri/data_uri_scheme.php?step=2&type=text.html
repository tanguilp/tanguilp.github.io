Données requises

Il faut, pour les ressources distantes ou pour du texte, indiquer le type MIME. Par exemple : "text/html", "image/png", "application/xhtml+xml;charset=iso-8859-1", etc.
Par défaut, le type MIME est "text/plain;charset=US-ASCII", donc si vous travaillez en utf-8, il vous faudra l’indiquer.
